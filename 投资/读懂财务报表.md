# 年报总述
+ 公司报表分一季度，半年度，三季度，年度四个周期报表，年度报表最详细可信，但依旧存在造假可能。
+ 年报必须包含资产负债表，现金流表，利润表，其中每个表又必须披露合并报表和母公司报表。 (参考: https://xueqiu.com/2504698885/119754481)
+ 会计审计意见: 财务报告需要市场上有审计资格的会计事务所出具审计报告，财务报告头部一般会存在会计事务所审计意见一般为“标准无保留意见”，表示会计事务所审计公司财务无问题(也可能没有发现问题)。非此类意见的报告存在大概率造假可能
+ 财务报表重要章节
    - 财务会计报告
    - 董事会报告
    - 重要事项
    - 股份变动
    - 股东情况

# 资产负债表
## 货币资金
+ 货币资金是流动资产的一种，属于流动性最强的资产。货币资金的种类也有很多，可以分为使用受限制和不受限制的货币资金
+ 公司获取货币资金的渠道有三种 1) 发行股票或者举债 2）出售资产或者业务部门 3）经营活动的现金流入
+ 货币资金需要与企业的短期债务与经营能力匹配。货币资金太少短期偿债能力薄弱，由资金流断裂的风险；货币资金过多说明企业资金运用能力不足
+ 财务报告中可能存在的货币资金风险:
    - 货币资金余额比短期负债小很多 (短期偿债能力差)
    - 货币资金充裕却借了很多有息债 (资金被挪用或者被冻结)
    - 其他资金很多，没有合理解释 （资金虚构）
## 经营相关资产
### 应收票据
+ 市场交易可能是现货现款交易也可能是应收应付票据交易，资产负债表中凡涉及增值税的企业，应收的票据，应收款项和现金都包含了增值税，但利润表剔除了增值税的部分
+ 应收票据可能是银行承兑汇票(可信度高)也可能是商业承兑汇票(可信度稍低)，考虑货币的时间价值(利息)承兑汇票的价值略低于等额现金。如果着急用钱，企业可以提前找银行兑付汇票，叫做贴现，代价是贴付一定利息，银行也可以找其他银行兑付，叫做再贴现
+ 应收票据的类型可以反映出企业的市场地位和销售策略，如果存在大量的应收票据或者商业承兑票占比很大，说明企业销售策略宽松；如果应收票承兑票数量和企业经营规模严重不符，可能企业在炒汇票
### 应收账款
+ 应收账款属于赊账，连票据都没有，应收账款能反应出企业的市场地位、风险、市场的竞争情况。多数商品属于买方市场，企业都会存在应收账款，但是数量要和经营规模匹配，且要足够稳定
+ 因为我国使用全职责发生制记账，应收账款会被计入资产，但实际上货物发出去了资金并没有收回来，应收账款越多越久，产生坏账的可能性越大；国家要求企业针对应收账款要进行计提坏账准备(根据应收账款年限在资产中缩减这份账款的账面价值)
+ 计提坏账准备的存在使得企业可以操纵利润表，因为计提坏账会减少当年利润，后续再收回利润会被补充回来，这就达到了控制利润的目的。计提坏账准备的方式很多，一般来说越严格的标准越可信，但是要注意计提标准突然发生变化
### 预付账款
+ 预付账款反应了企业在市场中的地位或者信用，预付账款的时间和数量都要注意，一般预付账款不会太久，不会有人原因提前很久给人钱却不收货
### 其他应收款
+ 与主营业务无关的应收款都会放在里面，它的属性天然决定了它是一个垃圾筐，也是最容易产生问题的地方。好的公司其他应收款一般很少或者分类明确
### 存货
+ 存货是由 原材料、在产品、库存商品、工人工资等构成。存货的风险在于有些商品有贬值风险，不同行业对商品有自己的计提标准，要注意核对。存货数量巨大也是比较大的风险
+ 利用存货可以对公司利润率造假，利润率 = (总利润 - 总成本) / 商品数量， 成本 = 固定成本 + 可变成本。因为固定成本不变，当生产的数量变多时，平均成本会被摊薄，单品利润率变高，这就是规模效应。但是规模效应的前提是生产的商品都能销售出去，如果只是生产而不销售，在报表里面看利润率也会变高，但是实际上并无任何作用

## 生产相关资产
### 固定资产
+ 固定资产是与公司经营相关价值比较大使用周期超过一年的非货币资产，一般包含厂房，设备，运输工具等
+ 固定资产每年需要进行折旧和减值准备(由于技术陈旧，损耗，闲置等导致实际价值低于账面价值)，都需要从利润中扣除
+ 折旧和减值准备方式很多，主要注意折旧方式的变化。好公司一般愿意快速折旧，减少当期利润从而达到前期少缴税后期多交税，相当于向税务局贷款
+ 折旧方式是固定的，但是实际上有些资产的实际折旧幅度没有那么大，这部分就是优质的资产

### 在建工程和工程物资
+ 在建工程建成后立即转为固定资产，在建工程不需要折旧，如果发现在建工程资产数量巨大切迟迟不转为固定资产，可能公司为了美化当期利润不想折旧，也可能这个在建工程本身就不存在

### 无形资产
+ 包括专利权，著作权，土地使用权，特许权，版权，非专利技术等等，和固定资产一样也需要每年折旧（摊销）

### 商誉
+ 如果一个企业的获利能力超过其可辨认资产的正常能力，多出来的这部分获利能力一定是另外的资产引起的，可能是管理方式，企业文化，品牌口碑，这个多出来的资产被称作商誉
+ 商誉本身的特殊性导致它的直面价值和实际价值相差很大，没有进行过收购的企业商誉为0，但是并不者意味着企业没有商誉。当企业A以1000万收购公允价值只有700万的公司，多出来的三百万就会被记录为商誉。但商誉也可能分文不值
### 长期待摊费用
+ 长期待摊费用是一种公司已经支出但是功效持续一年以上的费用，主要是对固定资产的修理和改良
+ 长期待摊费用已经被花掉， 它的实际效用不好界定，存在一些模糊边界，比如广告和研发投入就可以划入其中，长期待摊费用作为资产的一种可以影响利润，注意警惕公司将费用资产化
### 递延所得税资产和负债
+ 递延所得税是因为公司财务核算的利润和税务局核算的利润不一致产生一个概念。公司会计如果碰到税务局计算的利润比自己计算的多，为了报表数据统一，会认为多出来的这部分属于寄放在税务局的资产；反之则当做欠税务局的负债

## 投资相关资产
### 交易性金融资产
+ 如果公司把持有的债券、股票、基金等资产放进交易性金融资产，表示公司打算短期持有获取差价，这些股票基金的涨跌会作为公司当期损益，影响当期利润
### 持有至到期投资
+ 持有至到期投资一般是各类债券，
+ 金钱的时间价值: 因为利息的存在，同样的钱在今年和明年的价值不同，比如银行提供5%无风险利息，则今年的100万就是明年的105万，或者明年的105万就是今年的100万(折现)。央行加息导致股票市场下跌的一大原因就是因为，利率的上涨会导致企业未来的收益在当前的价值变低。比如一家公司在当前市场利率下五年后的收益为2000万，投资者对它现在的估值1000万。如果央行提高了利率，它未来还是可以赚2000万，但是现在的估值就不值1000万了

# 利润表
## 利润的重点
+ 股价 = 市盈率(PE) * 每股收益，所以投资可以从市盈率和每股收益两个角度分析。PE代表着市场对公司的前景预估，多少年投资可以回本;每股收益表示公司的盈利能力
+ 市盈率和每股收益都可以成为投资逻辑，所以需要同时考虑，市场情绪高涨可以成为投资的理由，但是需要判断每股收益会不会降低，否则两相抵消，股价不会上涨

## 资产负债表
+ 营业利润 = 收入-支出 (不考虑投资收益和资产折旧)
+ 利润总额 = 营业利润 + 主营业务之外的收入 (主营业务之外的收入容易造假)
+ 净利润 = 利润总额 - 所得税 
+ 上市公司可能存在子孙公司，母公司于子孙公司之间的关系比较复杂，法律上他们属于独立个体，实际上母公司可以通过大量控股和协议的方式控制子公司的商业行为。因为存在控股关系，子公司的部分或者全部权益和收益归母公司所有，也体现在母公司的利润表中

## 概念
+ 每股收益: 归属母公司所有者净利润 / 总股本
+ 市盈率(PE): 股票价格 / 每股收益。投资回报率的倒数，5PE代表投资回报率20%，意味着五年回本
+ 盈余公积金: 中国公司法规定公司盈利后必须保留10%盈利作为公积金，不可作为分红分配，是一种强制的扩大再生产的手段
